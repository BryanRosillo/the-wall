# Wall Page (UX Specification)

## States

### Loading state:
- Before phrases are displayed, a loading indicator is shown while phrases are being fetched.

## Layout & Content
- The wall displays a collection of phrases distributed across the screen.

Each phrase displays:
- Phrase text
- Author alias
- Visual style (font, size, color)

## Interaction
- The user can navigate the wall to explore phrases.
- A “Publish phrase” action is available from the wall page.

## Publish flow
When the user chooses to publish a phrase:
- A form is presented containing:
  - Text input
  - Font selection
  - Font size selection
  - Text color selection

After successful publication:
- The wall is refreshed.
- The newly published phrase is prominently displayed.
